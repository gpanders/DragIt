Search Engine URL: %s
Format must allow search terms (%s) to be added to the end URL

        


Lookup Search Engine URLs on the Mycroft Project

Example:
Mycroft Project lists Google US as: http://www.google.com/search?q=%s
The search terms (denoted by %s) are added at the end of the URL.
For this extension, remove %s and insert as Search Engine URL: http://www.google.com/search?q=
